# 🛒 Superstore Sales Analysis

## 📌 Project Overview
Analysis of a Global Superstore Sales dataset to uncover business insights 
on sales trends, profit, customer segments, and regional performance.

## 📊 Dataset
- **Source:** Kaggle — Superstore Sales Dataset
- **Records:** 9,994 orders
- **Columns:** Order Date, Sales, Profit, Category, Region, Segment, Discount

## 🔍 Key Insights
- 📈 Technology category has the highest sales
- 📉 Furniture has the lowest profit margin
- 🌍 West region leads in total sales
- 👥 Consumer segment contributes 50%+ of revenue
